# UAE CV Builder

A mobile-friendly CV builder web application designed for the UAE context. Users can fill in personal details, upload a profile picture, select designations, choose job descriptions, preview their CV, and download it as a PDF.

## Features

- Input personal details: Name, Date of Birth, Nationality, Passport Number & Expiry, UAE Visa Status & Expiry.
- Select designation from predefined options or enter a custom one.
- Dynamic job description selection based on the chosen designation.
- Upload a profile picture or capture via mobile camera.
- Live CV preview.
- Download CV as a PDF file.

## Blue-Collar Professions Included

- Electrician
- Plumber
- Construction Laborer
- Carpenter
- Cleaner / Janitor
- Driver
- Security Guard

*(Each profession has at least three job descriptions.)*
